---
name: project-create
description: Generates a SYSMOD project.

Use when the user requests creating a new SYSMOD project. 
---

When generating a SYSMOD project:

Create the SysML v2 textual notation for the project as specified in the SYSMOD SysML v2 library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml. The project occurrence definition in the SYSMOD library include metadata @AI which provides a create_prompt. Use the prompt.
