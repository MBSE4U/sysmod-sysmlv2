---
name: brownfield-architecture-create
description: Generates a SYSMOD brownfield-architecture.

Use when the user requests creating a new SYSMOD brownfield-architecture. 
---

When generating a SYSMOD brownfield-architecture:

Create the SysML v2 textual notation for the brownfield-architecture as specified in the SYSMOD SysML v2 library https://github.com/MBSE4U/sysmod-sysmlv2/blob/main/SYSMOD.sysml. The brownfield-architecture part usage in the SYSMOD project definition in the SYSMOD library include metadata @AI which provides a create_prompt. Use the prompt.
